## 2024-05-15 - Multiple Derived Values Loop Iteration
**Learning:** In Flutter apps, when calculating multiple derived values (e.g., counts for different statuses) from a single list of objects (like `members`), calling `.where().length` multiple times creates redundant iterations over the list. This happens a lot in dashboards or lists calculating statistics.
**Action:** Replace multiple chained `.where().length` or similar iteration methods with a single manual iteration (like a `for` loop with a `switch` statement or `if-else` blocks) to accumulate the counts simultaneously. Also cache any dynamically evaluated parameters passed to the loop (such as `DateTime.now()`) and ensure expensive object-level method calls (like `.getStatus()`) are only executed once per object.
